# Mumble2

# Installation
* 1 - clone repo https://github.com/victorsmart99998/mumble.git
* 2 - Create an account on agora.io and create an app to generate an APP ID
* 3 - Update APP ID, Temp Token and Channel Name in room_rtc.js
